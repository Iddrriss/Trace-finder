# Trace-finder
A professional, modular forensic triage tool for Windows systems, TraceFinder helps incident responders, digital forensic examiners, and security analysts quickly detect user activity within a configurable time window (default: 180 minutes). Built with forensic best practices and zero external dependencies.


## Features
Execution Evidence - UserAssist, Prefetch files
File Activity - Recent files, RecentDocs registry
Network Activity - Chrome/Edge/Firefox browser history, downloads
Hardware Tracking - USB device connection history
Command Line - PowerShell history, Run dialog (RunMRU)
Registry Artifacts - TypedPaths (Explorer address bar)
## Requirements
None!
## Installation & Usage
Clone the repository:
git clone https://github.com/Iddrriss/Trace-finder.git
cd Trace-finder
Run the script:
python3 tracefinder.py
Happy Forensics!
